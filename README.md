# Instructions

## 1. Model Training & Evaluation Metrics
Navigate to the scripts branch and execute the `virtual_mouse.ipynb` Python notebook
to train a `.task` model using the gesture dataset and evaluate it on several classification metrics.

## 2. Virtual Mouse Test
Execute the `virtual_mouse_test.py` script to launch the virtual mouse. Current gestures include
- Mouse Cursor: Index Finger Point
- Left-Click: Touch Index Finger to Thumb Finger
- Right-Click: Touch Middle Finger to Thumb Finger
